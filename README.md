# 🎬 Netflix Content Analytics & Strategic Insights

## 📌 Overview

This project presents a comprehensive analysis of Netflix's content catalog by integrating multiple public datasets to uncover trends in content production, audience preferences, genre distribution, and release patterns. The project also features an interactive dashboard that enables users to explore insights through visual analytics.

---

## 🎯 Objectives

- Analyze Netflix's global content catalog.
- Identify trends in movies and TV shows.
- Study genre, country, and release-year distributions.
- Compare audience ratings using IMDb and Netflix datasets.
- Generate business insights through interactive visualizations.

5. Update the dataset paths in the notebook and run all cells.
