# 📌 Project Overview

Netflix has evolved into one of the world's largest streaming platforms with content spanning hundreds of countries and thousands of titles. This project performs comprehensive exploratory data analysis by integrating multiple public datasets to analyze content growth, audience preferences, genre popularity, production trends, ratings, and geographic distribution.

The project also includes an interactive executive dashboard that enables users to explore insights through intuitive visualizations.

---

# 💼 Business Problem

Media companies require data-driven insights to understand:

* Content production trends
* Audience preferences
* Genre popularity
* Geographic expansion
* Content ratings
* Release strategies

This project transforms raw entertainment datasets into actionable business intelligence to support strategic decision-making.

---

# 🎯 Project Objectives

* Analyze Netflix's global content catalog.
* Study Movies vs TV Shows distribution.
* Discover genre popularity trends.
* Analyze content production by country.
* Explore content addition trends over time.
* Compare IMDb ratings and Netflix metadata.
* Build an executive dashboard for interactive analytics.

# 📈 Key Business Insights

* Netflix's catalog spans **23,000+ titles** across **147 countries**, highlighting strong global expansion.
* Movies account for the majority of the catalog, while TV Shows represent a growing share.
* International markets contribute significantly to recent content additions.
* Drama, Comedy, and Documentary remain the most dominant genres.
* Content production accelerated rapidly after 2015.
* TV-MA is the most common maturity rating.
* Geographic analysis reveals the leading countries driving Netflix's content production.
* Interactive dashboards enable stakeholders to identify trends, explore catalogs, and support strategic decision-making.

# ▶️ How to Run

1. Clone the repository.
2. Install all required dependencies.
3. Download the datasets from the links above.
4. Place the datasets inside your working directory.
5. Open the notebook in Google Colab or Jupyter Notebook.
6. Run all cells sequentially.
7. Explore the interactive dashboard for business insights.
